# Installing Git

## On Windows

To install Git on Windows, you can [download it here](https://git-scm.com/download/win).

## On Mac (Homebrew)

For Mac, using `Homebrew`:

```sh
brew install git
```

## On Linux

For Linux, it depends on your package manager (based on your distribution).

### Using APT

```sh
sudo apt-get install git
```

### Using Pacman

```sh
sudo pacman -S git
```
